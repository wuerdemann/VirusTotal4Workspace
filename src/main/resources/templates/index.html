VirusTotal for IBM Watson Workspace

Introduction and Terms of Use

Introduction

This Bot provides virus "scanning" for files that are added to a IBM Watson Workspace conversation.

Therefore this bot sends the SHA-256 hash of an attached file to VirusTotal. If the file is known to be malicious, the user gets a notification. If the state is unknown, the user may decide to send the whole file to VirusTotal

NB: Sending files to VirusTotal will disclose confidential content and VirusTotal may use and redistribute your data! (see VirusTotal Rights)
